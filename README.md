# HomeStock Pro

**HomeStock Pro** — это программа для отслеживания ваших вещей: что где лежит, куда вы это положили и в каком количестве.

- **Назначение:** каталогизация вещей и контроль их количества  
- **Хранение данных:** все данные сохраняются локально в браузере, интернет не требуется  
- **Конфиденциальность:** информация не отправляется на серверы, всё остаётся на вашем устройстве  

## Установка и запуск

1. **Скачайте** проект или откройте страницу приложения.
2. **Откройте** файл в браузере HomeStock.html.
3. Приложение сразу готово к работе, установка не требуется.

HomeStock Pro помогает быстро находить вещи и не забывать, где что лежит.
