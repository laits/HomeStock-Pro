## 📌 Описание

**Что где находится?** — это простое одностраничное приложение для каталогизации домашних вещей.  
Оно помогает быстро находить предметы, помнить, где они лежат, и отслеживать их количество.

Все данные хранятся **локально в браузере**, интернет не требуется.

---

## ✨ Возможности

- 📁 Каталогизация вещей по названию, категории, месту и расположению  
- 📝 Заметки к каждой вещи (например, «докупить», «проверить», и т.п.)  
- 🖼️ Добавление фотографий (с автоматическим сжатием, до 10 МБ)  
- 🔍 Поиск по названию, категории, месту и заметке  
- 🔢 Учёт количества  
- 🏷️ Генерация QR‑кодов для маркировки  
- 📊 Просмотр статистики (объём хранилища и количество записей)  
- 🌗 Переключение темы (светлая/тёмная, если включено в приложении)  
- 🔐 Полностью локальная работа без отправки данных на сервер
- 🗃️ Архивирование данных вещей с возможностью переноса на другое устроиство

---

## 🔒 Конфиденциальность

- Все данные сохраняются в **LocalStorage** вашего браузера  
- Никакая информация **не отправляется** на удалённые серверы  
- Приложение работает **полностью офлайн**

## 🚀 Установка и запуск

1. **Скачайте** файл `HomeStock.html`.  
2. **Откройте** его в любом современном браузере (Chrome, Firefox, Edge, Safari и т.п.).  
3. Приложение сразу готово к работе — установка не требуется.
